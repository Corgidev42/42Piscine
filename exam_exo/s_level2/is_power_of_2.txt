Nom de l'exercice : is_power_of_2
Fichiers attendus : is_power_of_2.c
Fonctions autorisées : Aucune
--------------------------------------------------------------------------------

Écrire une fonction qui détermine si un nombre donné est une puissance de 2.

Cette fonction doit retourner 1 si le nombre donné est une puissance de 2, sinon elle retourne 0.

Votre fonction doit être déclarée comme suit :
int		is_power_of_2(unsigned int n);
